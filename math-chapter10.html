Trapezoidal Rule

Piecewise-linear approximation · Coefficient pattern 1,2,…,2,1 · O(h²) error

EXPLAIN

The composite trapezoidal rule approximates ∫ₐᵇ f(x) dx by dividing [a, b] into N equal subintervals of width h = (b−a)/N, then connecting adjacent function values with straight lines. The area of each resulting trapezoid is (h/2)(f(xₖ) + f(xₖ₊₁)). Summing all N trapezoids, each interior node appears twice and gets weight 2; the two endpoints get weight 1.

  • ∫ₐᵇ f(x) dx ≈ (h/2) [f₀ + 2f₁ + 2f₂ + … + 2fₙ₋₁ + fₙ]
  • h = (b−a)/N — N subintervals produce N+1 equally-spaced nodes
  • Endpoints f₀ and fₙ → weight 1 (each belongs to one trapezoid)
  • Interior nodes f₁ … fₙ₋₁ → weight 2 (each shared by two adjacent trapezoids)
  • Error order: O(h²) — halving h reduces truncation error by a factor of 4
  • No constraint on N — works for any N ≥ 1, even or odd
  • Works when f is only known as a table of values (no analytic formula required)
WORKED EXAMPLE

Approximate ∫₀² x² dx using the trapezoidal rule with N = 4 subintervals.

1 Find h:   h = (b − a) / N = (2 − 0) / 4 = 0.5
2 List nodes:   x₀ = 0.0 · x₁ = 0.5 · x₂ = 1.0 · x₃ = 1.5 · x₄ = 2.0
3 Compute f(xₖ) = xₖ²:   f₀ = 0 · f₁ = 0.25 · f₂ = 1.0 · f₃ = 2.25 · f₄ = 4.0
4 Sum interior nodes:   f₁ + f₂ + f₃ = 0.25 + 1.0 + 2.25 = 3.50
5 Apply formula:   (h/2)[f₀ + 2(3.50) + f₄] = 0.25 × [0 + 7.0 + 4.0] = 0.25 × 11.0 = 2.75

Exact value: ∫₀² x² dx = [x³/3]₀² = 8/3 ≈ 2.667  ·  Absolute error ≈ 0.083  ·  With N = 8 (h = 0.25): error ≈ 0.021 — confirms the factor-of-4 reduction from O(h²).

THINK OF IT LIKE…

Paint rollers of different widths. One wide roller pressed across a curved wall surface — the flat roller misses or overshoots the curve everywhere. Switch to ten narrow rollers placed edge-to-edge: their collection of flat tops approximates the curve much more closely. Each roller is one trapezoid; the width of each roller is h. The key insight: narrower rollers (smaller h) → error shrinks as h² → much better approximation.

Simpson's 1/3 Rule

Parabolic approximation · Pattern 1,4,2,4,…,4,1 · N must be even · O(h⁴) error

Complete Section 1 to unlock

EXPLAIN

Simpson's 1/3 rule fits a parabola (quadratic polynomial) through every triplet of equally-spaced nodes and integrates it exactly — one parabola per two subintervals. This extra curvature-fitting makes it far more accurate than the trapezoidal rule at the same step size: error drops from O(h²) to O(h⁴). The constraint is that N must be even so that the interval can be divided into complete triplet groups.

  • ∫ₐᵇ f(x) dx ≈ (h/3) [f₀ + 4f₁ + 2f₂ + 4f₃ + 2f₄ + … + 4fₙ₋₁ + fₙ]
  • Coefficient pattern: 1 · 4 · 2 · 4 · 2 · … · 4 · 1
  • Endpoints f₀ and fₙ → weight 1
  • Middle of each parabola group → weight 4 (controls the bend)
  • Shared node between two groups → weight 2 (right end of one, left end of next)
  • N must be even — each parabola spans 2 subintervals (3 nodes)
  • Error order: O(h⁴) — halving h reduces error by factor of 16

WORKED EXAMPLE

Approximate ∫₀¹ eˣ dx using Simpson's 1/3 rule with N = 4 subintervals.

1 Check N:   N = 4 is even ✓ → Simpson's rule applies
2 Find h:   h = (1 − 0) / 4 = 0.25
3 Nodes and f(xₖ) = eˣₖ:   f₀ = 1.0000 · f₁ ≈ 1.2840 · f₂ ≈ 1.6487 · f₃ ≈ 2.1170 · f₄ ≈ 2.7183
4 Apply coefficients (1, 4, 2, 4, 1):   1.0000 + 4(1.2840) + 2(1.6487) + 4(2.1170) + 2.7183 = 1.0000 + 5.1360 + 3.2974 + 8.4680 + 2.7183 = 20.6197
5 Multiply by h/3:   (0.25/3) × 20.6197 = 0.08333 × 20.6197 ≈ 1.7183

Exact: e − 1 ≈ 1.71828  ·  Error ≈ 0.00002  ·  Trapezoidal (same N = 4): ≈ 1.7272, error ≈ 0.009 — Simpson's is ~450× more accurate.

THINK OF IT LIKE…

Ruler vs flexible curve template. The trapezoidal rule uses a rigid ruler to connect two dots with a straight line — it ignores everything between. Simpson's rule uses a flexible drafting curve that bends through three dots at once. The flexible template captures the curvature of f(x) that straight lines cannot see. The extra bending degree (quadratic vs linear) makes the area estimate dramatically more accurate — at the cost of one rule: you always need an even number of intervals so the template always has a middle point to bend through.

Error Analysis & Method Comparison

O(h²) vs O(h⁴) · When to use which · Absolute error calculation · Method selection

Complete Section 2 to unlock

EXPLAIN

Choosing between the two methods requires comparing their accuracy vs constraints. Simpson's rule is always more accurate for smooth functions at the same step size, but it demands an even N. The trapezoidal rule is universally applicable. Understanding the error orders lets you predict — without re-running — how much accuracy you gain by refining h.

  • Same h, smooth f: Simpson's error ≪ trapezoidal error (e.g. ~450× for eˣ at N = 4)
  • Halving h: Trapezoidal error × (1/4) · Simpson's error × (1/16)
  • When to use trapezoidal: N is odd · data is non-smooth · quick baseline estimate needed
  • When to use Simpson's: N is even · f is smooth · maximum accuracy at given N is required
  • Absolute error = |approximation − exact value|
  • Practical O(h²) test: double N → if error ratio ≈ 4, it's O(h²)
  • Practical O(h⁴) test: double N → if error ratio ≈ 16, it's O(h⁴)

WORKED EXAMPLE — Error Comparison

Compare the error of both methods for ∫₀¹ eˣ dx with N = 2, then predict the error for N = 4.

1 Exact value:   ∫₀¹ eˣ dx = e − 1 ≈ 1.71828
2 Trapezoidal (N = 2, h = 0.5):   (0.5/2)[1 + 2e^0.5 + e] ≈ 0.25[1 + 3.2974 + 2.7183] = 0.25 × 7.0157 ≈ 1.7539  →  Error ≈ |1.7539 − 1.71828| = 0.0356
3 Simpson's (N = 2, h = 0.5):   (0.5/3)[1 + 4e^0.5 + e] ≈ (0.5/3)[1 + 6.5949 + 2.7183] = (0.5/3) × 10.3132 ≈ 1.7189  →  Error ≈ |1.7189 − 1.71828| = 0.0006
4 Predict N = 4 errors (halving h):   Trapezoidal: 0.0356 ÷ 4 ≈ 0.0089  ·  Simpson's: 0.0006 ÷ 16 ≈ 0.00004

At N = 2: Trapezoidal error (0.0356) is ~60× larger than Simpson's (0.0006). At N = 4: Trapezoidal ≈ 0.009 (verified above), Simpson's ≈ 0.00002 — the accuracy gap widens as h shrinks.

THINK OF IT LIKE…

Speed cameras vs radar guns. The trapezoidal rule is a fixed-point speed camera — it samples speed at two moments and draws a straight line between them. Simple, universally installable (any N), but misses acceleration curves. Simpson's rule is a radar gun tracking the car's actual curved trajectory through three time points — far more precise at the same sampling rate. But the radar needs readings in pairs; if you have an odd leftover reading, fall back to the speed camera. This maps exactly to the N-even constraint: when you can form pairs, use Simpson's; when you can't, use trapezoidal.
